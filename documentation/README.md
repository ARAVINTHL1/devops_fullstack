# Dharani Jewellery Mobile Application - Complete Documentation

## Overview

This documentation contains the complete project report for the **Dharani Jewellery Mobile Application** - a comprehensive business management system for small-scale jewellery shops.

The application is designed to digitalize and streamline the internal operations of jewellery businesses through a modern, secure, and user-friendly mobile platform built with React, Node.js, and MongoDB.
