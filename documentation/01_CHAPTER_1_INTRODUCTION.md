# CHAPTER 1: INTRODUCTION & SYSTEM OVERVIEW

## 1.1 Introduction

The Dharani Jewellery App is a smart business management solution developed to simplify and enhance the daily operations of a jewellery shop.

### Problem Statement

In traditional jewellery businesses, managing product inventory, tracking stock levels, and maintaining sales records are often handled manually or with basic digital tools, which can lead to:
- Inefficiencies in operations
- Data inconsistencies
- Lack of proper tracking and monitoring
- Time-consuming manual processes

### Solution Overview

This application addresses these challenges by providing:
- **Centralized digital platform** - For shop owner and workers to manage tasks efficiently
- **Real-time operations** - Web-based application converted to mobile for improved accessibility
- **Secure access control** - Only authorized users (owner and staff) can access the system
- **Modern technology stack** - React (frontend), Node.js (backend), MongoDB (database)

### Key Functionalities

The application provides:
- Product management
- Inventory tracking and stock management
- Sales recording and monitoring
- Real-time data handling
- Easy accessibility on mobile devices
- Employee operations management

### Technology Stack

| Component | Technology |
|-----------|-----------|
| Frontend | React.js |
| Backend | Node.js |
| Database | MongoDB |
| Mobile | Mobile-responsive React |
| Integration | WhatsApp API |

## 1.2 Existing System

### Current Challenges in Traditional Jewellery Shops

Many small-scale jewellery businesses like Dharani Jewellery rely on manual methods or basic digital tools for daily operations, leading to:

#### Manual Record Maintenance
- Handwritten records or basic spreadsheets used for inventory and sales
- Increases human error chances
- Risk of data loss
- Time-consuming record retrieval

#### Lack of Real-Time Updates
- Absence of real-time synchronization of stock and sales data
- Workers and owner cannot access updated information instantly
- Delayed decision-making
- Inaccurate stock levels

#### Inefficient Inventory Management
- Manual tracking leads to overstocking or stock shortages
- No automated system to monitor inventory
- Poor stock optimization
- Wastage and financial losses

#### Limited Accessibility
- Traditional systems not mobile-friendly
- Cannot access information from different devices
- Restricted flexibility in managing operations
- Limited remote management capabilities

#### Poor Data Organization
- Lack of structured system makes retrieval difficult
- Time-consuming to analyze business performance
- Unreliable data for decision-making
- Difficulty in auditing and compliance

#### Absence of Secure Access Control
- No proper authentication mechanisms
- Data accessible without security restrictions
- Risk of unauthorized modifications
- Loss of data integrity

## 1.3 System Study

The Dharani Jewellery App is designed to overcome limitations of traditional systems by providing a centralized, efficient, and user-friendly digital solution.

### System Components

#### User Interface
- Developed using **React** for responsive and intuitive interface
- Simple and clean design for easy navigation
- Mobile-friendly interface
- Features: product management, stock updates, sales records
- Easy access for both owner and workers

#### Backend and Data Management
- **Node.js** handles application logic and user requests
- **MongoDB** stores product details, stock information, sales records
- Centralized data storage with real-time access
- Structured data organization
- Reduces inconsistencies and improves accuracy

#### Authentication and Security
- Secure login mechanism with valid credentials
- Restricts access to authorized users only (owner and workers)
- Protects sensitive business data
- Prevents unauthorized modifications
- Session management support

#### Product and Inventory Management
- Add, update, and delete jewellery products easily
- Real-time stock level tracking
- Monitor inventory status
- Prevent stock shortages or overstocking
- Efficient resource control

#### Sales Management
- Record daily sales transactions
- Maintain accurate sales data
- Track business performance
- Retrieve past sales information
- Financial record management

#### Additional Features
- WhatsApp integration for customer communication
- Analytics and reporting
- Loan management for credit-based purchases
- Dashboard with business insights
